# meetUPPER CLI

## About
Node.js CLI for Meetup.

## How to Install
First, install the CLI by npm:
```bash
$ npm install -g meet-upper-cli
```

## Commands
#### Display incoming events for group
```bash
$ meet-upper-cli UPCOMING <group>
```
